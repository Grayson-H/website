All About PC's



Parts Needed To Build A PC


  • Graphics Card (GPU)
  • Processor (CPU)
  • Motherboard
  • Memory (RAM)
  • Power Supply
  • WiFi Card (If going to use wifi)
  • Case

How to build a PC

  1. Open the case

  2. Screw motherboard into the case

  3. Mount CPU
    1. Locate the CPU socket holder on the motherboard
    2. Lift up the latch lever to release and hinge open the socket cover
    3. Holding the CPU by its sides, line up any allignment notches or the triangle on the corner of the CPU to the triangle marked on the motherboard to ensure the correct orientation
    4. Gently place it straight down into the motherboard socket to seat the CPU

  4. Install CPU cooler
    1. If required, apply thermal paste to the back of the CPU
    2. Seat CPU cooler and fix in position
    3. Plug the power cable attached to the cooler fan into the motherboard connector
